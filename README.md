## How to install

- With computer:

Use [Sideloadly](https://sideloadly.io/#download "Sideloadly")
> AltStore has issues with the iPAs for some reason.

<br>

- Without computer:

1. Use [eSign](https://github.com/iOS17/Esign "eSign")

2. Use [kSign](https://www.reddit.com/r/sideloaded/comments/1n172fi/full_guide_on_how_to_sideload_with_ksign_using/ "kSign")

3. Use [Scarlet](https://usescarlet.com/ "Scarlet") [Less reliable due to being unsigned a lot]

## Issues

- Remember when Logging into the app with 2FA enabled in the Username field type "Username|2FA"
- iPad commonly has issues with Korone iPAs, if you try joining a game and get a black screen, it doesn't work.
- 2021 has been having issues a lot, not just on iOS. If you get an error joining a 2021 game keep trying until it lets you, or try a different game.
<br>

> "I get a "Network Error" on the Avatar Page

- AVATAR EDITOR DOES NOT WORK ON MOBILE!!!!!!!!!!!!!!!
<br>

> "This game failed to fetch join script."

- Make sure your actually joining a 2021 game by going to https://pekora.zip/games
<br>

> Failed to connect to the Game. (ID = 17: Connection attempt failed.)

- Try using a VPN like [Cloudflare WARP](https://1.1.1.1 "Cloudflare WARP") or [ProtonVPN](https://protonvpn.com/ "ProtonVPN")
<br>

> "Something went wrong. Please try again later." on Sign Up screen

- DO NOT SIGN UP ON THE APP!!!! SIGN UP AT https://pekora.zip/auth/home
